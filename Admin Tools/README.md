# Admin Tools

This folder contains Tampermonkey userscripts and utilities for Canvas LMS administrators and instructional designers at **Northern Kentucky University**.

Each sub-folder is a self-contained plugin with its own README, installation instructions, and version history.

---

## Available Plugins

| Plugin | Description |
|---|---|
| [Canvas-Enrollment-Plugin](./Canvas-Enrollment-Plugin/) | Adds an Enrollment Management panel to Canvas course pages — enroll/unenroll as Designer, validate links, check due dates, and monitor Canvas status |

---

## Installation

All plugins are Tampermonkey userscripts. Install [Tampermonkey](https://www.tampermonkey.net/) first, then follow the installation link in each plugin's README.
