# Admin Tools

Tampermonkey userscripts and utilities for Canvas administrators and instructional design staff at **Northern Kentucky University**.

---

## Tools

| Tool | Description |
|---|---|
| [Canvas Enrollment Plugin](./Canvas-Enrollment-Plugin/) | Adds an Enrollment Management panel to Canvas course pages — enroll/unenroll yourself as a Designer and run course-health checks |

---

## Getting Started

All tools in this folder are [Tampermonkey](https://www.tampermonkey.net/) userscripts. To use them:

1. Install the Tampermonkey browser extension.
2. Open the `README.md` inside the tool's folder and follow the one-click install link.
3. Navigate to the relevant Canvas page — the script activates automatically.
