# Instructor Tools

This folder contains Tampermonkey userscripts and utilities for Canvas LMS instructors and teaching assistants at **Northern Kentucky University**.

Each sub-folder is a self-contained plugin with its own README, installation instructions, and version history.

---

## Available Plugins

| Plugin | Description |
|---|---|
| [Canvas-Instructor-Plugin](./Canvas-Instructor-Plugin/) | Adds an Instructor Tools panel to Canvas course pages — link validator, due-date checks, grade weighting diagnostic, and CETI appointment booking (NKU only) |

---

## Installation

All plugins are Tampermonkey userscripts. Install [Tampermonkey](https://www.tampermonkey.net/) first, then follow the installation link in each plugin's README.
