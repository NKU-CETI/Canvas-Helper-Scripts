# Browser Extensions

Sideloadable browser extensions for NKU Canvas administrators and helpdesk staff. These extensions provide the same functionality as the Tampermonkey userscripts but can be loaded directly into Chrome or Edge without requiring Tampermonkey.

## Available Extensions

| Extension | Description |
|---|---|
| [NKU-Canvas-Toolkit](./NKU-Canvas-Toolkit/) | Combined admin and helpdesk panel. Enrolls as Designer (CETI role) or Helpdesk (Enroll Help Desk role), runs course health checks, and diagnoses module completion issues — all in a single role-gated panel. |

## Quick Install (Chrome / Edge)

1. Open `chrome://extensions` (or `edge://extensions`).
2. Enable **Developer mode**.
3. Click **Load unpacked** and select the extension's folder (e.g. `NKU-Canvas-Toolkit`).

See each extension's own `README.md` for full installation instructions, role requirements, and update guidance.
